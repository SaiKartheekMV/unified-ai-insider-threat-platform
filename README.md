# Unified AI-Driven Insider Threat Detection & Zero-Trust Access Platform

## Overview
This project aims to detect insider threats in enterprise systems using AI-based behavioral analysis while enforcing Zero-Trust security principles.

## Problem Statement
Traditional security systems fail to detect malicious activities performed by legitimate users. Insider threats remain one of the most expensive and undetected security risks.

## Solution
An AI-powered platform that:
- Continuously monitors user behavior
- Detects anomalies using ML models
- Enforces zero-trust access decisions
- Provides real-time alerts and analytics

## Tech Stack
- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: Node.js, Express, PostgreSQL, Redis
- AI Engine: Python, FastAPI, Isolation Forest
- DevOps: Docker, GitHub Actions
